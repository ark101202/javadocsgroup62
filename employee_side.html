employee_side Class

Employee_side class handles the employee-side functionality for a JavaFX application. It connects to a PostgreSQL database to manage and display product information.

Methods

  • initialize()

    Initializes the employee_side controller by setting up the TableView and connecting to the database.

  • closeWindow()

    Closes the application window.

  • addProduct()

    Adds a new product to the TableView.

  • removeProduct()

    Removes the selected product from the TableView.
